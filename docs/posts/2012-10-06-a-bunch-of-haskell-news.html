A bunch of Haskell news

A few quick informations:

  • Manuel Chakravarty is running a poll on community interest in a potential book covering advanced Haskell topics. Be quick to fill it out - it ends on Monday!

  • GHC 7.6.1 was released exactly a month ago. See release notes for more details. Next version of GHC is scheduled for release in late 2012/early 2013 (according to this video).

  • New release of Haskell Communities and Activities Report is scheduled for November. Contact Janis Voigtländer if you have something to contribute.

  • After a very long break, new issue of The Monad.Reader was released in the end of August. It contains my “Understanding Basic Haskell Error Messages” tutorial aimed for beginners.
